# Telegram Bot with Node.js & Telegraf

بوت تيليجرام بسيط بلغة Node.js باستخدام مكتبة telegraf.

## تشغيل محلياً

1. ثبت الحزم:
   ```
   npm install
   ```

2. أنشئ ملف `.env` وضع فيه التوكن:
   ```
   BOT_TOKEN=ضع_هنا_توكن_البوت
   ```

3. شغل البوت:
   ```
   npm start
   ```

## نشر على Render

- ارفع الملفات إلى GitHub.
- أنشئ خدمة Node.js جديدة في Render.
- أضف متغير بيئة باسم `BOT_TOKEN` وضع فيه التوكن.
- لا حاجة لأي إعدادات خاصة، كل شيء سيعمل تلقائياً.
